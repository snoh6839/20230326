$tHeight = 5;
for ($i = 1; $i <= $tHeight; $i++) {
for ($j = 1; $j <= $i; $j++) {
echo "*";
}
echo "<\n>";
}
?>

이 코드는 다음과 같이 동작합니다.

1. 바깥쪽 for 루프에서는 $i 변수를 1부터 5까지 1씩 증가시키며, 5번 반복합니다. 즉, 각 줄마다 해당하는 $i 변수의 값이 달라지게 됩니다.

2. $i 변수는 현재 찍을 별의 줄을 나타내며, 첫 번째 루프에서 $i가 1일 때는 첫 번째 줄에 별을 찍게 됩니다.

3. 안쪽 for 루프에서는 $j 변수를 1부터 $i까지 1씩 증가시키며, $i번 반복합니다. 즉, 해당하는 줄에서 찍을 별의 개수만큼 반복하게 됩니다.

4. $j 변수는 현재 줄에서 찍을 별의 개수를 나타내며, 예를 들어 $i가 3일 때는 3번 반복하며 한 줄에 ***을 찍습니다.

5. 안쪽 for 루프에서 별 찍기가 끝나면, 바깥쪽 for 루프에서 줄바꿈을 수행합니다. 즉, 각 줄이 끝난 후에 줄바꿈을 하게 됩니다.
